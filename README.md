## 🗓️ Lộ trình 6 ngày

<details>
<summary>📅 <strong>Ngày 1: Git Cơ bản</strong></summary>

### Nội dung học:
- Git là gì và tại sao cần dùng Git?
- Cài đặt và cấu hình Git
- Repository, Working Directory, Staging Area
- Các lệnh cơ bản: `init`, `add`, `commit`, `status`

### Bài tập:
- Tạo repository đầu tiên
- Thực hiện commit đầu tiên
- Kiểm tra trạng thái repository

### Thời gian: ~2 giờ
</details>

<details>
<summary>📅 <strong>Ngày 2: Làm việc với Repository</strong></summary>

### Nội dung học:
- Xem lịch sử commit với `log`
- Di chuyển giữa các commit
- Hoàn tác thay đổi
- Git ignore và .gitignore

### Bài tập:
- Tạo .gitignore cho dự án
- Thực hiện rollback commit
- Xem chi tiết lịch sử thay đổi

### Thời gian: ~2.5 giờ
</details>

<details>
<summary>📅 <strong>Ngày 3: Branching & Merging</strong></summary>

### Nội dung học:
- Khái niệm Branch
- Tạo và chuyển đổi branch
- Merge và Rebase
- Giải quyết conflict

### Bài tập:
- Tạo feature branch
- Merge branch vào main
- Giải quyết merge conflict

### Thời gian: ~3 giờ
</details>

<details>
<summary>📅 <strong>Ngày 4: GitHub Cơ bản</strong></summary>

### Nội dung học:
- GitHub là gì?
- Tạo repository trên GitHub
- Push/Pull code
- Clone repository

### Bài tập:
- Tạo GitHub repository
- Push dự án lên GitHub
- Clone repository về máy

### Thời gian: ~2 giờ
</details>

<details>
<summary>📅 <strong>Ngày 5: Collaboration</strong></summary>

### Nội dung học:
- Fork repository
- Pull Request workflow
- Code Review
- Issues và Project management

### Bài tập:
- Fork một dự án open source
- Tạo Pull Request
- Review code của teammate

### Thời gian: ~3.5 giờ
</details>

<details>
<summary>📅 <strong>Ngày 6: Git Nâng cao</strong></summary>

### Nội dung học:
- Git hooks
- Submodules
- Git workflows (Gitflow, GitHub Flow)
- Best practices

### Bài tập:
- Setup Git hooks
- Implement Gitflow workflow
- Optimize Git workflow

### Thời gian: ~4 giờ
</details>

## 🚀 Bắt đầu nhanh

### 📋 Yêu cầu hệ thống
- Node.js 18.0+ 
- npm/yarn/pnpm
- Git đã được cài đặt

### 🛠️ Cài đặt

```bash
# Clone repository
git clone https://github.com/yourusername/git-github-roadmap.git

# Di chuyển vào thư mục dự án
cd git-github-roadmap

# Cài đặt dependencies
npm install
# hoặc
yarn install
# hoặc
pnpm install

# Chạy development server
npm run dev
# hoặc
yarn dev
# hoặc
pnpm dev
```

Mở [http://localhost:3000](http://localhost:3000) để xem kết quả.

### 🌐 Deploy

**Vercel (Recommended):**
```bash
npm install -g vercel
vercel --prod
```

**Netlify:**
```bash
npm run build
# Upload dist folder to Netlify
```

## 🤝 Đóng góp

Chúng tôi hoan nghênh mọi đóng góp! 

### Cách đóng góp:
1. **Fork** repository này
2. **Tạo branch** cho feature: `git checkout -b feature/AmazingFeature`
3. **Commit** thay đổi: `git commit -m 'Add some AmazingFeature'`
4. **Push** lên branch: `git push origin feature/AmazingFeature`
5. **Mở Pull Request**

### Guidelines:
- Tuân thủ coding standards
- Viết tests cho features mới
- Cập nhật documentation
- Sử dụng conventional commits
